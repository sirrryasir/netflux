# Netflux

A high-fidelity streaming platform clone built with **React**, **Tailwind CSS**, and **TMDB API**. This project is designed for educational purposes, specifically for intermediate students in the Ardaykaab Frontend Bootcamp.

## Features
- **Landing Page**: Professional Netflix-inspired UI with interactive FAQ and Trending sections.
- **Dynamic Data**: Real-time movie data fetched from the TMDB API using the native `fetch()` API.
- **Multi-Page Routing**: Seamless navigation between Landing, Login, and Browse pages using `react-router-dom`.
- **Interactive Modals**: Detailed movie information view with smooth animations and high-fidelity design.
- **Responsive Design**: Fully mobile-responsive layout using Tailwind CSS Grid and Flexbox.

## Built With
- **React (Vite)**: Modern frontend library.
- **Tailwind CSS**: Utility-first CSS framework.
- **React Router**: Client-side routing.
- **TMDB API**: Real-world movie database.
- **Lucide React**: Beautiful icons.

## Getting Started

### Prerequisites
- Node.js installed on your machine.
- A TMDB API Key (Get yours at [themoviedb.org](https://www.themoviedb.org/)).

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/netflux.git
   ```
2. Navigate to the project directory:
   ```bash
   cd netflux
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```

## Curriculum Structure (For Instructors)
This project is broken down into 4 logical phases to help students learn progressively:
1. **Phase 1**: Foundation & Static UI (Components & Layout).
2. **Phase 2**: Navigation & Routing (Multi-page flow).
3. **Phase 3**: API Integration (Live data fetching).
4. **Phase 4**: Premium Features (Modals & Polish).
